SkillPrep AI Agent — Starter Scaffold

This repository is a starter scaffold for the SkillPrep AI Agent platform (fullstack).

Structure:
- `backend/` — Node.js + Express API, Mongoose models, AI agent stubs
- `frontend/` — React + Vite starter with Tailwind placeholder

Quick start (Windows PowerShell):

```powershell
npm install
npm run dev:backend    # runs backend in dev (nodemon)
npm run start:frontend  # runs frontend dev server (Vite)
```

See the `backend` and `frontend` folders for implementation notes and environment examples.
